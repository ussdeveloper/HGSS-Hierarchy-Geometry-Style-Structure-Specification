## [1.0.0] - 2025-11-13

### Added
- Initial release of HGSS specification
- JSON schema for validation
- Support for hierarchical structure with Groups and Features
- CSS-like styling system with cascade priority
- Label positioning with coordinates and centroid calculation
- Bidirectional conversion with GeoJSON, KML, GML, TopoJSON
- JavaScript utility functions
- Comprehensive documentation and examples
- Open source license (CC-BY-SA 4.0)
